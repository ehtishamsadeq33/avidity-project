In my existing React Native / Expo app with Node.js backend, implement PDF generation using Puppeteer on the server. Do exactly the following:
PART 1 — Backend setup:

Install puppeteer on the backend:

npm install puppeteer

Create a new file artifacts/api-server/src/routes/pdf.ts
Add a POST endpoint POST /api/report/pdf that:

Accepts the full report data in the request body
Generates a PDF using Puppeteer
Returns the PDF as a base64 string



typescriptimport puppeteer from 'puppeteer';

router.post('/pdf', async (req, res) => {
  try {
    const { candidateInfo, perceivedActual, composites, scores } = req.body;

    const html = generateReportHTML(candidateInfo, perceivedActual, composites, scores);

    const browser = await puppeteer.launch({
      args: ['--no-sandbox', '--disable-setuid-sandbox'],
      headless: true
    });

    const page = await browser.newPage();
    await page.setContent(html, { waitUntil: 'networkidle0' });
    await page.waitForTimeout(1000);

    const pdf = await page.pdf({
      format: 'A4',
      printBackground: true,
      margin: { top: '20px', bottom: '20px', left: '20px', right: '20px' }
    });

    await browser.close();

    const base64 = Buffer.from(pdf).toString('base64');
    res.json({ pdf: base64 });
  } catch (error) {
    res.status(500).json({ error: 'PDF generation failed' });
  }
});

For the pie chart use this canvas code inside the HTML:
html<canvas id="pieChart" width="300" height="300"></canvas>
<script>
  const scores = SCORES_DATA;
  const colors = ['#e8e83a','#c8d439','#8fbc3b','#4da63e','#2e8b44','#a0522d','#c65b22','#e07000','#d44000','#b22222'];
  const canvas = document.getElementById('pieChart');
  const ctx = canvas.getContext('2d');
  const total = Object.values(scores).reduce((a,b) => a+b, 0);
  let startAngle = -Math.PI / 2;
  Object.entries(scores).forEach(([score, count], i) => {
    if (count === 0) return;
    const sliceAngle = (count / total) * 2 * Math.PI;
    ctx.beginPath();
    ctx.moveTo(150, 150);
    ctx.arc(150, 150, 130, startAngle, startAngle + sliceAngle);
    ctx.closePath();
    ctx.fillStyle = colors[i];
    ctx.fill();
    startAngle += sliceAngle;
  });
</script>

In ReportPage3, replace the current PDF generation code with:
javascriptimport * as FileSystem from 'expo-file-system';
import * as Sharing from 'expo-sharing';

const downloadPDF = async () => {
  try {
    setGeneratingPDF(true);

    const response = await fetch(`${API_BASE_URL}/api/report/pdf`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        candidateInfo,
        perceivedActual,
        composites,
        scores
      })
    });

    const { pdf } = await response.json();

    // Save base64 PDF to device
    const fileUri = FileSystem.documentDirectory + `CRA_Report_${candidateInfo.name}.pdf`;
    await FileSystem.writeAsStringAsync(fileUri, pdf, {
      encoding: FileSystem.EncodingType.Base64
    });

    // Share/download the PDF
    await Sharing.shareAsync(fileUri, {
      mimeType: 'application/pdf',
      dialogTitle: 'Download CRA Report',
      UTI: 'com.adobe.pdf'
    });

  } catch (error) {
    Alert.alert('Error', 'Could not generate PDF. Please try again.');
  } finally {
    setGeneratingPDF(false);
  }
};
