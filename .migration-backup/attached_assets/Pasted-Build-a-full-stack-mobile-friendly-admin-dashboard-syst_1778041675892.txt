## ⚙️ BACKEND REQUIREMENTS

Create these APIs:

### Auth

POST /admin/login
GET /admin/me

### Dashboard

GET /admin/stats
→ returns:

* total submissions
* weekly/monthly counts
* average scores

GET /admin/candidates
→ supports query params:

* search
* filters
* pagination

GET /admin/candidate/:id
→ full data

GET /admin/report/:id/pdf
→ generate PDF

GET /admin/export/csv
